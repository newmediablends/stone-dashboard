## Why this is a project

"AI as Second Brain" has been flagged as In Progress on the 10x learning queue since April 13, but it had no container and no arc. This project gives it both. The frame comes from Hilary Gridley's Lenny's Podcast episode — the "anti-system" philosophy — which matches how Mike already thinks about systems: minimal configuration, maximum observation, 10x gate on everything.

This is not about rebuilding what works. It's about closing the specific gaps between what Mike has (deep intelligence layer, PARA, team, memory) and what the anti-system playbook asks for (frictionless capture, observation loop, narrate-before-automate, small first actions).

---

## North Star

> A personal operating system that refuses to be set up — AI observes how Mike actually works, writes down what it sees, and gets better on its own. Capture near-zero-friction. Automations earned by repeated pain. Every decision runs through the 10x lens.

Four load-bearing principles:
1. Observation beats configuration
2. Friction-zero capture
3. 10x gate on every automation
4. Small first actions (25–30 min max)
